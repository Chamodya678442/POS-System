Project title: Point-Of-Sales(POS) System

How to run the application
1. This assignment is implemented using netBeans IDE, if this run using netBeans IDE extract the zip file and open that folder using it.
2. Open the project named "POS system" inside the folder.
3. Before run this application, make sure that there is MySQL connector is already added to the libraries otherwise it can not run correctly.
4. First, run the file named as "LoginForm.java" (Directory: POS_System>Source Packages>default packages>LoginForm.java)with your credentials (username and password)
5. After login correctly, there are two options "Manage products" and "Products and new sale".
6. In "Manage products", a new products can be added to the database.
7. After a new sale process create a bill according to the sales product can be done using "Products and new sale". And also it reduces the product quantities in the database according to the sale.
